##Omnivore

Feed caching with Redis and Sinatra

##Dependencies

* ruby >= 1.9
* thin
* sinatra
* redis
* redis gem (https://github.com/redis/redis-rb)
* rest-client (https://github.com/rest-client/rest-client)
* rspec
* rack-test
* timecop
