Hi. I'm omnivore, your friendly feed cache server.

To request a feed hit!

/feed?url={feed_url} #returns xml

To get the metadata for a feed strike!

/feed_data?url={feed_url} #returns json

If you want the feed along with its metadata wrapped up in one tasty object add "include_feed=true" to the feed data request's GET parameters. Kabam!

/feed_data?url={feed_url}&include_feed=true
