Hi, I'm Omnivore.

To request a response

/feed?url={url} #returns xml

To get the metadata for a response

/feed_data?url={url} #returns json

If you want the response along with its metadata wrapped up in one object add "include_feed=true" to the request's GET params.

/feed_data?url={feed_url}&include_feed=true

To view application configs go to /configs (configs are editable in omnivore_config.yml)

To get the data for all stored responses (not for production :) ) !

/all_feed_data
