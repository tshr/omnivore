Hi, I'm Omnivore.

To request a response

/request?url={url} #returns xml

To get the metadata for a request

/request_data?url={url} #returns json

If you want the response along with the request's metadata add "include_response=true" to the request's GET params.

/request_data?url={url}&include_response=true

To view application configs go to /configs (configs are editable in omnivore_config.yml)

To get the data for all stored responses (not for production :) ) !

/all_requests
