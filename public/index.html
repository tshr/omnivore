Hi. I'm Omnivore, your friendly cache server.

To request a response hit!

/feed?url={feed_url} #returns xml

To get the metadata for a response strike!

/feed_data?url={feed_url} #returns json

If you want the response along with its metadata wrapped up in one tasty object add "include_feed=true" to the request's GET parameters. Kabam!

/feed_data?url={feed_url}&include_feed=true

Last, to view applications configs go to /configs PEW PEW PEW!
